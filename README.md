## 部署到 GitHub Pages

### 第一步：创建仓库

1. 在 GitHub 上创建一个新仓库（例如 `zuiwengting`）
2. 将本目录下所有文件推送到该仓库的 `main` 分支

### 第二步：启用 GitHub Pages

1. 进入仓库的 **Settings** 页面
2. 左侧菜单选择 **Pages**
3. **Source** 选择 **Deploy from a branch**
4. **Branch** 选择 `main`，**Folder** 选择 `/(root)`
5. 点击 **Save**

### 第三步：等待部署

约 1-2 分钟后，访问 `https://你的用户名.github.io/你的仓库名/` 即可看到网站。

### 注意事项

- 本站点为纯静态网站，**无需任何服务器端配置**
- 所有资源路径均为相对路径 `./`，支持子目录部署
- 视频文件较大（约 3MB），首次加载可能需要等待
- 推荐在 Chrome / Safari / Firefox 现代浏览器中浏览
